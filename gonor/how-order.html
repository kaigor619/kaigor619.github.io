Как заказать?

( из каталога товаров )

1. выберите раздел с товарами, который вас интересует:

2. выберите из каталога товар, который вам больше всего понравился и нажмите на него:

3. на странице с товаром проделайте следующее:

  • выберите параметры товара

    ( тип материала - чехлы, чашки; размер - футболки, Свитшоты )

  • нажмите кнопку «купить», после чего вас перенаправит на страницу с оформлением заказа, где вам необходимо заполнить личные данные для отправки заказа:

4. после оформления заказа ожидайте Сообщение/звонок от нашего менеджера для подтверждения Заказа

Как заказать?

( свой дизайн )

1. для заказа вещи со своим дизайном вам нужно нажать кнопку «создать свой дизайн» и оформить заявку на создание макета:

2. после оформления заявки наш дизайнер свяжется с вами и отправит макет, а также обсудит детали:

3. оформить заказ вы можете прямо в диалоге с нашими дизайнерами и в течени нескольких дней ваш заказ будет уже у вас.
